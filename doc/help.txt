Usage:
  kimquery [options]

Description:
  Command-line interface for retrieving data from OpenKIM.

Options:
  -h, --help            output help information
  -f, --format          output format, currently support 'csv' and 'json'
  -u, --update          update cache
  -p, --props [args]    specify properties needed, separated by ','
                        support regular expressions
                        use '-p list' to show supported properties
                        modify prop_meta.json to support more properties
  -s, --structs [args]  specify properties needed, separated by ','
                        use '-p list' to show supported properties
  -e, --elems [args]    specify properties needed, separated by ','
  -m, --models [args]   specify properties needed, separated by ','
                        support regular expressions
